# LXW AI Studio

这是一个可以直接部署到 GitHub Pages 的个人技术服务静态网站。

## 项目文件

- `index.html`：页面结构
- `style.css`：网站样式与响应式布局
- `script.js`：导航交互、滚动高亮、进入视口动画
- `README.md`：项目说明与部署步骤

## 本地预览

直接双击 `index.html`，或使用浏览器打开即可预览。

## 部署到 GitHub Pages

当前仓库地址：

`https://github.com/wsllsw1/lsw`

推荐设置：

1. 进入仓库的 `Settings`
2. 打开左侧 `Pages`
3. 在 `Build and deployment` 中选择 `Deploy from a branch`
4. `Branch` 选择 `main`
5. 文件夹选择 `/ (root)`
6. 点击 `Save`

等待几十秒到几分钟后，网站地址应为：

`https://wsllsw1.github.io/lsw/`

## 自定义内容建议

部署前可以按需替换以下占位信息：

- Telegram：`@yourname`
- Email：`your@email.com`
- 微信文案：`请联系获取`

## 适用场景

- 个人接单主页
- AI 服务展示页
- 个人技术名片网站
- 自由职业服务落地页
