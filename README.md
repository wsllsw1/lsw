# LXW 资源导航

这是一个部署在 GitHub Pages 上的个人资源分享 / 工具导航静态网站。

当前定位：

- 资源分享
- 工具导航
- 实用网站收藏
- 个人软件和教程入口

后续可以继续扩展：

- 添加自己的软件作品
- 添加软件下载地址
- 添加教程文章
- 添加资源分类页或搜索数据文件

## 项目文件

- `index.html`：页面结构和资源内容
- `style.css`：网站样式与响应式布局
- `script.js`：导航菜单、搜索、分类筛选和滚动高亮
- `README.md`：项目说明与部署说明

## 本地预览

直接双击 `index.html`，或使用浏览器打开即可预览。

## GitHub Pages

仓库地址：

`https://github.com/wsllsw1/lsw`

线上地址：

`https://wsllsw1.github.io/lsw/`

Pages 推荐设置：

1. 进入仓库的 `Settings`
2. 打开左侧 `Pages`
3. 在 `Build and deployment` 中选择 `Deploy from a branch`
4. `Branch` 选择 `main`
5. 文件夹选择 `/ (root)`
6. 点击 `Save`

## 内容维护

资源卡片目前写在 `index.html` 中。后续如果资源变多，可以改成从 JSON 文件读取，再自动渲染分类和卡片。
